# Project Overview

TravelGo is a responsive travel website UI designed in Figma to provide users with an engaging platform for exploring destinations, booking trips, and planning travel. The design adapts seamlessly across desktop and mobile devices while maintaining a consistent user experience.

---

# Objectives

- Design a modern and responsive travel website.
- Create layouts for both desktop and mobile devices.
- Improve usability through intuitive navigation.
- Deliver a visually appealing travel booking interface.
